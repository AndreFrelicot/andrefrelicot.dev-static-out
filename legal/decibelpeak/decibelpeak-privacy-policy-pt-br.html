Política de Privacidade

DATA DE VIGÊNCIA: 22 de setembro de 2025
ÚLTIMA ATUALIZAÇÃO: 22 de setembro de 2025

DEFINIÇÕES

Para os fins desta Política de Privacidade:

  • "Aplicativo" ou "App" refere-se ao Decibel Peak, o aplicativo de software desenvolvido por André Frélicot.
  • "Desenvolvedor", "Nós", "Nosso" refere-se a André Frélicot.
  • "Usuário", "Você", "Seu" refere-se ao indivíduo que acessa ou usa o Aplicativo.
  • "Dados Pessoais" significa qualquer informação relacionada a uma pessoa física identificada ou identificável.
  • "Processamento" significa qualquer operação realizada em dados pessoais, incluindo coleta, armazenamento, uso ou divulgação.

1. ESCOPO E APLICAÇÃO

1.1. Esta Política de Privacidade rege a coleta, uso e divulgação de informações pelo aplicativo Decibel Peak para plataformas iOS e macOS.

1.2. Ao baixar, instalar ou usar este Aplicativo, você reconhece que leu, compreendeu e concorda em ser vinculado por esta Política de Privacidade.

1.3. O Desenvolvedor não coleta, armazena, processa ou transmite quaisquer dados pessoais.

2. COLETA DE INFORMAÇÕES

2.1. Nenhuma Coleta de Dados Pessoais: O Desenvolvedor declara explicitamente que nenhum dado pessoal é coletado, armazenado, processado ou transmitido por este Aplicativo.

2.2. As seguintes categorias de informações NÃO são coletadas:

  1. Identificadores pessoais (incluindo, mas não limitado a nomes, endereços de e-mail, números de telefone, endereços postais);
  2. Identificadores de dispositivo (incluindo, mas não limitado a identificadores únicos de dispositivo, identificadores de publicidade, endereços MAC);
  3. Dados de localização (incluindo, mas não limitado a coordenadas GPS, endereços IP que poderiam revelar localização);
  4. Análises de uso ou dados comportamentais;
  5. Relatórios de crash contendo informações pessoais;
  6. Gravações de áudio ou dados de áudio armazenados;
  7. Histórico de medições ou leituras armazenadas.

3. PROCESSAMENTO DE ÁUDIO

3.1. Processamento em Tempo Real Apenas: O Aplicativo acessa o microfone do dispositivo exclusivamente para fins de medição de níveis sonoros em tempo real.

3.2. Os dados de áudio são processados de acordo com os seguintes princípios:

  1. Fluxos de áudio são processados imediatamente após o recebimento;
  2. Nenhum dado de áudio é gravado, armazenado ou em cache;
  3. Nenhum dado de áudio é transmitido para servidores ou serviços externos;
  4. Dados de áudio são descartados imediatamente após o cálculo de decibéis;
  5. Nenhum armazenamento persistente de informações relacionadas ao áudio ocorre.

4. USO E PROCESSAMENTO DE DADOS

4.1. Dado que nenhum dado pessoal é coletado, não há dados pessoais para usar, processar, analisar ou manipular de outra forma.

4.2. O Aplicativo opera inteiramente através de processamento local no dispositivo do usuário.

5. COMPARTILHAMENTO E DIVULGAÇÃO DE INFORMAÇÕES

5.1. Nenhum Compartilhamento de Informações: O Desenvolvedor não compartilha, vende, aluga ou divulga de outra forma quaisquer informações a terceiros, pois nenhuma informação é coletada.

5.2. O Aplicativo não se integra com quaisquer serviços de terceiros que coletam dados pessoais, incluindo, mas não limitado a:

  1. Serviços de análise;
  2. Redes de publicidade;
  3. Plataformas de mídia social;
  4. Serviços de armazenamento em nuvem;
  5. Serviços de relatório de crash que coletam dados pessoais.

6. ARMAZENAMENTO E SEGURANÇA DE DADOS

6.1. Nenhum Armazenamento de Dados: Nenhum dado pessoal é armazenado em servidores controlados pelo Desenvolvedor.

6.2. Nenhum dado pessoal é armazenado no dispositivo do usuário além das preferências padrão do aplicativo que não contêm informações pessoais.

6.3. Como nenhum dado pessoal sensível é coletado ou armazenado, medidas tradicionais de segurança de dados para dados pessoais não são aplicáveis.

7. DIREITOS E PERMISSÕES DO USUÁRIO

7.1. Acesso ao Microfone

7.1.1. O Aplicativo requer acesso ao microfone para funcionar adequadamente.

7.1.2. Os usuários podem conceder ou negar permissão do microfone através das configurações do dispositivo.

7.1.3. A permissão do microfone pode ser revogada a qualquer momento através das configurações do sistema iOS/macOS.

7.1.4. O Aplicativo não funcionará sem acesso ao microfone.

7.2. Direitos do Titular dos Dados

7.2.1. Sob as leis de privacidade aplicáveis (LGPD, GDPR, CCPA, etc.), os usuários têm certos direitos em relação aos dados pessoais. No entanto, como nenhum dado pessoal é coletado:

  1. Direito de Acesso: Nenhum dado pessoal existe para acessar;
  2. Direito de Exclusão: Nenhum dado pessoal existe para excluir;
  3. Direito de Retificação: Nenhum dado pessoal existe para corrigir;
  4. Direito de Portabilidade: Nenhum dado pessoal existe para exportar;
  5. Direito de Oposição: Nenhum processamento de dados ao qual se opor.

8. PRIVACIDADE DE CRIANÇAS

8.1. O Aplicativo é compatível com a Lei Geral de Proteção de Dados (LGPD) e Children's Online Privacy Protection Act (COPPA) por design, pois nenhuma informação pessoal é coletada de usuários de qualquer idade.

8.2. Nenhuma informação pessoal é coletada de crianças menores de 13 anos.

8.3. O Aplicativo não requer registro de usuário, criação de conta ou qualquer forma de submissão de informações pessoais.

9. TRANSFERÊNCIAS INTERNACIONAIS DE DADOS

9.1. Nenhuma transferência internacional de dados ocorre, pois nenhum dado pessoal é coletado ou transmitido.

9.2. Todo o processamento ocorre localmente no dispositivo do usuário.

10. TECNOLOGIAS DE RASTREAMENTO

10.1. O Aplicativo não emprega quaisquer tecnologias de rastreamento, incluindo, mas não limitado a:

  1. Cookies;
  2. Web beacons;
  3. Pixels de rastreamento;
  4. Mecanismos de rastreamento entre sites;
  5. Identificadores de publicidade.

11. BASE LEGAL PARA PROCESSAMENTO (LGPD/GDPR)

11.1. Sob a Lei Geral de Proteção de Dados (LGPD) e o Regulamento Geral sobre a Proteção de Dados (GDPR), nenhuma base legal para processamento de dados pessoais é necessária, pois nenhum dado pessoal é processado.

11.2. O acesso ao microfone é concedido através do consentimento explícito do usuário via prompts de permissão do sistema iOS/macOS.

12. ALTERAÇÕES NESTA POLÍTICA DE PRIVACIDADE

12.1. O Desenvolvedor reserva-se o direito de modificar esta Política de Privacidade a qualquer momento.

12.2. Alterações materiais serão refletidas atualizando a data "Última Atualização" no início deste documento.

12.3. Dado que nenhum dado pessoal é coletado, as alterações abordarão principalmente:

  1. Esclarecimentos das práticas de privacidade existentes;
  2. Atualizações das informações de contato;
  3. Conformidade com regulamentações de privacidade novas ou alteradas.

13. CONFORMIDADE REGULATÓRIA

13.1. Esta Política de Privacidade é projetada para estar em conformidade com as leis e regulamentações de privacidade aplicáveis, incluindo:

  1. Lei Geral de Proteção de Dados (LGPD) - Brasil;
  2. Regulamento Geral sobre a Proteção de Dados (GDPR) - União Europeia;
  3. California Consumer Privacy Act (CCPA) - Califórnia, Estados Unidos;
  4. Children's Online Privacy Protection Act (COPPA) - Estados Unidos;
  5. Requisitos de privacidade da Apple App Store;
  6. Outras leis regionais de privacidade aplicáveis.
